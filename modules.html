Project Modules

This project has declared the following modules:

Name Description
Fulcrum Security API Fulcrum Security Basic API
Fulcrum Security Hibernate Impl Fulcrum Security Hibernate Impl is an implementation of the Fulcrum Security API that uses Hibernate as a database backend.
Fulcrum Security Memory Impl Fulcrum Security Memory Impl is an implementation of the Fulcrum Security API that uses simple HashMaps as a backend. It is meant mainly for test purposes.
Fulcrum Security Torque Impl Fulcrum Security Torque Impl is an implementation of the Fulcrum Security API that uses Apache DB Torque as a database backend.
Fulcrum Security Distribution Build The parent POM for the Apache Turbine and Fulcrum project.
